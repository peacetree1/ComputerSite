- Spring Boot로 구현한 컴퓨터 판매 게시판 입니다.
- 제작기간 (2023.06.23 ~ 2023.07.11)

- sts4 버전 2.7.13
- Java 11
- MySQL DB
- MyBatis
- MVC패턴
